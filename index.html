UFC Rivals

App Web Full Stack - Proyecto Final

Una plataforma web integral para el análisis y simulación de peleas de UFC con integración de IA. Incluye gestión de usuarios, autenticación y predicciones interactivas de combates.

Características Principales:

  • Arquitectura MVC con Thymeleaf para renderizado dinámico del lado del servidor.
  • Autenticación segura con Spring Security
  • Integración con OpenAI API para simulaciones interactivas de combates
  • Frontend responsive con visualización de estadísticas
  • Sistema de alertas y notificaciones en tiempo real

Tecnologías:

Spring Boot Spring Security PostgreSQL OpenAI API Tailwind CSS Chart.js SweetAlert

Completado

E-Commerce Sales Forecast

Proyecto Fin de Máster

abr. 2026 – jul. 2026 • Asociado con Monlau Formación Profesional

Pipeline completo de Ciencia de Datos enfocado en analítica predictiva y comportamiento del consumidor sobre un dataset transaccional de más de 540K registros.

Características Principales:

  • Ingeniería de Datos: limpieza, filtrado de cancelaciones y estructuración de variables temporales.
  • Modelado Predictivo: regresión temporal con LightGBM, XGBoost y Random Forest mediante ventana deslizante.
  • Clustering y Perfilado: detección de anomalías con Isolation Forest, reducción con PCA y segmentación con K-Means.
  • Visualización: dashboards interactivos en Power BI para comparativa de métricas y análisis de clústeres.

Tecnologías:

Python Pandas Polars Scikit-Learn LightGBM XGBoost Power BI
